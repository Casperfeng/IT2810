Trykk for dokumentasjon

CSS Grid / Flexbox:

CSS Grid og Flexbox er verktøy man kan benytte seg av for å oppnå dynamiske layouts. CSS Grid er et to dimensjonelt system og kan dele inn i både rader og kolonner, mens Flexbox egner seg best for inndeling i en dimensjon enten rad eller kolonne.


I mitt prosjekt har jeg valgt å benytte meg av Flexbox fordi det passet det tenkte designet for nettsiden som jeg ønsket å gjøre så simpelt som mulig. Jeg har benyttet meg av flex-containers for å midtstille innholdet, og for å skalere med skjermstørrelse.

HTML5 Canvas / SVG:

Canvas er et HTML element (container) som benyttes for å tegne grafikk på nettsider. For å tegne benyttes Javascript gjennom et eget Canvas API.


SVG er også et HTML element (container) som benyttes for å fremstille grafikk, men grafikken blir ikke tegnet ved Javascript. Det blir her benyttet et xml-format. SVG grafikk er vektorbasert og kan skaleres opp og ned uten å miste kvalitet. SVG elementer legger seg også i DOM, og man kan benytte seg av ulike event handlers for å manipulere grafikken, eller benytte kun ren css.


Her har jeg benyttet meg av SVG og Canvas på to ulike kunstverk med mål om å ha samme funksjonalitet for begge. Dette gjorde jeg for å bli bedre kjent med begge alternativene. Om jeg skulle valgt et av alternativene for å bruke videre, hadde jeg gått for SVG. Dette er fordi det skalerer bedre, og er bedre bygget for animasjoner. Måten jeg har brukt Canvas på fører til at alle figurene må bli tegnet på nytt flere ganger for å oppnå animasjon.

jQuery:

JQuery er et lite og raskt Javascript-biblotek som inneholder funksjonalitet for DOM-manipulasjon og event-handling. JQuery var veldig populært før i tiden når browser-compatibility kunne by på problemer.


I dette prosjektet har jeg brukt JQuery for å selektere DOM-elementer. Dette er gjort for å manipulere elementene og legge til listeners ved hover, click og mouse-out eventer. JQuery brukes til å bygge alle elementene i Canvas kunstverket, animere bevegelsene, endre farger og å åpne og lukke dokumentasjon.

Cross-browser testing:

Cross-browser testing benyttes for å teste applikasjoner på tvers av ulike nettlesere.


For å teste nettsiden min testet jeg først all funksjonalitet og skalerbarhet i Chrome. Deretter testet jeg i Safari, og sørget for at nettsiden hadde ønsket utseende og oppførsel i begge disse.
